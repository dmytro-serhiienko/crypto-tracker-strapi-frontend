# Crypto Tracker

Crypto Tracker is a mobile-first Ionic Vue app for browsing cryptocurrency data, tracking favorites, and viewing basic market stats.

## Features

- Browse coins with search, filters, and sorting
- Open coin details with price and sparkline chart
- Save favorite coins for quick access
- View a simple stats screen with key market highlights

## Tech Stack

- Vue 3
- Ionic Vue
- Pinia
- Vite
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
